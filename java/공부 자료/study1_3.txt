	2. 그니까 변수명에는 동사를 쓰지마세요 최대한
	customer
	ex 뭔가를 세는 변수 cnt_consumer, consumer_num
	3. 변수명에 제발 전치사좀 쓰지 마세요. of
	4. 단수형 복수형 제발 구분좀 해주세요. customer_list
	5. 상수로 쓸거면 제발 대문자로 쓰세요. => 안바뀌는값 명시 필수
	6. 너무길어지면 적당히 잘라주세요. information => info
	함수는 14~15글자 변수명은 8글자 넘어가면 잘라 주세요.
	7. 상수는 길어도됩니다.
